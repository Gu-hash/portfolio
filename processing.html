Processing project

Pixel Art Project

This is a something I made as part of coding bootcamp during the
first year of university. It transforms one image into another by colouring
pixels based on the target image. The placement of the mouse determines which
pixels are coloured.I used Processing to create it.


When the space bar is pressed, the size ofthe pixels get bigger and when 'W'
is pressed, the pixels get smaller. Ialso included an extra feature so that
when 'X' is pressed and held down,the pixel shape turns from circles into squares.
